# AI Study Planner API

A backend API built with **FastAPI** that generates personalized study plans based on user-defined tasks, deadlines, priorities, and weekly availability. The system is designed to support AI-assisted planning while maintaining strict validation, predictable behavior, and clean API contracts.

## Features
- RESTful API built with FastAPI
- Health check endpoint for service monitoring
- Automatic OpenAPI / Swagger documentation
- Structured JSON request and response handling
- Designed for future integration with databases and AI planning services

## Tech Stack
- Python
- FastAPI
- Uvicorn
- JSON / OpenAPI
